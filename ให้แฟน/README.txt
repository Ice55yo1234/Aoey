วิธีใช้งานโปรเจกต์ "ของขวัญวันครบรอบ"

1. ติดตั้ง Flask
   pip install flask

2. เข้าไปที่โฟลเดอร์นี้
   cd "c:\Games\AI_CCTV\ให้แฟน"

3. รันเซิร์ฟเวอร์
   python server.py

4. เปิดเว็บเบราว์เซอร์บนคอมพิวเตอร์หรือมือถือในเครือข่ายเดียวกัน
   http://<IP ของคอมพิวเตอร์>:5000
   เช่น http://192.168.1.10:5000

   ถ้าไม่ทราบ IP ให้รันคำสั่งใน PowerShell:
   ipconfig
   แล้วดูค่า IPv4 Address

ในหน้าเว็บจะมีปุ่มให้กดดูข้อความพิเศษและบทกวีสำหรับวันครบรอบ
